The Treasure Map

This application compares craigslist free stuff postings by location. The distance of a free thing is directly related to its value. There are many treasures to be found, using this application, hopefully, you will find something near you worth giving a new home. Find it here.

It is written with the Flask Python framework, and the source code (MIT) can be found here.
